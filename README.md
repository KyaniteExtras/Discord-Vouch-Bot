# Features
 - **Scrapevouches** | Gather Vouches in a server (WIP)
 - **Addvouch** | Remotely add a vouch to a user
 - **Vouch** | Vouch A User
 - **Rep** | Check How Many Vouches a user has
 - **Vouches** | List who has vouched the user
